# Server Performance Stats Script (Beginner DevOps Project)

## Project Overview
This project provides a basic server performance monitoring script written in Bash.  
It helps system administrators or DevOps beginners quickly check key server resource usage.

---

##  Script Features
| Feature | Description |
|--------|-------------|
| CPU Usage | Shows user, system and idle CPU percentages |
| Memory Usage | Displays used, free, and percentage usage |
| Disk Usage | Shows total, used, and free disk space |
| Top Processes | Lists top 5 processes by CPU and Memory usage |
| System Info | Shows OS version, uptime, load average, and logged-in users |
